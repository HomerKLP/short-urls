## Инструкция по запуску на локале

- Склонировать проект с гита
- Скопировать **env** файл и настроить его
- Запустить docker-compose 
- Посмотреть логи запущенного контейнера, успешные логи выглядят так:

```log
2020/10/12 16:40:26 Sirius-short-urls server started...
2020/10/12 16:40:26 Connected to Database...
2020/10/12 16:40:26 Index initialized...
```

## Инструкция по запуску на проде

*Смотреть в репке **sirius-nginx**, потому что сервис коротких ссылок входит в состав первичных сервисов, которые запускаются из одного места*

## API документация

- `POST /api/generate-link/`

> Создание короткой ссылки

**PAYLOAD:**

```js
{
    "original_url": "https://google.kz/",   // Оригинальная ссылка
    "is_reusable": true                     // Многоразовая
}
```

- `GET /{token}`

> Переход по короткой ссылке

**PATH_PARAMS:**

```js
{
    "token": "7xC20K"   // Уникальный токен короткой ссылки
}
```
